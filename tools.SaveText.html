tools.SaveText

Save text or stream to a file, given a path.

Options

As a string, this is the encoding. Otherwise, it has the same meaning as the argument in the fs.readFile(filename[, options], callback) function of NodeJS.

You can specify the path attribute to it. In this case, the corresponding input is not definined.

Inputs

  • path {string}: path of the file to write. This input is not defined if options.pathis defined.
  • text {string}: text to store in this file.

Outputs

  • path {string}: text stored in this file.
