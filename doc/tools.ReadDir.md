# tools.ReadDir


Return an array of files' names contained in the folder `path`.

## Inputs
* __path__ {string}: path from where to list existing files.

## Outputs
* __files__ {array[string]}: array of the  names of the files in the directory excluding `'.'` and `'..'`.
