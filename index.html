Matis 0.1.0

Multi-level Asynchrounous Tools Interconnection System.

Fast complex asynchronous processes made easy

Concepts

  • Input driven execution: processing starts as soon as input is ready.
  • Parallelism: tools are executed in parallel if it gives a time benefit. The result is a very fast process.
  • Inputs order preserved: even if a tool processes 3 inputs at the same time, outputs are post to the next tool in the same order.
  • Tracability: you always know which input has been needed to produce any output.

Built-in tools

  • BaseName: basename of a path.
  • BlackHole: swallow garbaged outputs.
  • ChangeExtension: change extension of a filename.
  • ConcatStrings: concat many strings.
  • ConsoleLog: log messages for debug.
  • Constant: outputs a constant.
  • DirName: dirname of a path.
  • ExistsFile: test file existence.
  • ForEach: loop on each element of an array.
  • Join: join elements of an array.
  • LoadStream: load file as a stream.
  • LoadText: load file as a string.
  • MatchRegexp: match a string against a regular expression.
  • Nop: No operation.
  • PrefixSuffix: surround a string with two constants.
  • ReadDir: list all files and subdirectories in a directory.
  • SaveText: store string into a file.
  • ShellExec: execute a shell command.
